# MRI-based-Brain-Tumor-Detection
This project is an AI-based web application that detects brain tumors from MRI scan images using the VGG16 deep learning model. The system first preprocesses the MRI image by resizing and normalizing it. Then, the pre-trained VGG16 model analyzes the image and classifies it into one of four categories:
Glioma Tumor
Meningioma Tumor
Pituitary Tumor
No Tumor
The application is built using Python, TensorFlow/Keras, OpenCV, and Flask. Users can upload an MRI image through the web interface, and the system predicts the tumor type along with a confidence score. This helps provide quick and accurate assistance in brain tumor diagnosis.
